# FileFlow

フォルダ構造を可視化し、一括処理（拡張子付与など）を行ってZIP化するツール。
クライアントサイド(JS)のみで完結し、オフラインで安全に動作します。

## 主な仕様

*   **処理モード**: ファイルに対して `.md` または `.txt` 拡張子を付与（テキストファイル判定ロジックに基づく）。
*   **テキスト判定ロジック**:
    1.  **既知のバイナリ除外**: `.png`, `.exe`, `.zip` など、明らかにバイナリである拡張子は即座にスキップします（高速化のため）。
    2.  **Nullバイトチェック**: ファイルの先頭512バイトを読み込み、`0x00` (Null文字) が含まれるかチェックします。含まれていればバイナリとみなし、除外します。
    3.  **判定**: 上記を通過したものをテキストファイルとして扱います。これにより、未知の拡張子や拡張子なしのファイルも高精度に判定可能です。
*   **パフォーマンス**: 大量のファイル（2万ファイル以上）でもブラウザがフリーズしないよう、バッチ処理で非同期に実行されます。
*   **ZIP出力**: 
    *   単一フォルダ時は「フォルダ名.zip」
    *   複数時は「files.zip」
*   **設定**: ドットファイル（`.git`等）の除外（デフォルトON）。

## セットアップ（初回のみ）

外部ライブラリ(`jszip`)をローカルに配置するため、以下を実行してください。

```bash
sh setup.sh
```
